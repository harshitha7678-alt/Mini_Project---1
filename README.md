# 📊 Group DNA - WhatsApp Chat Analyzer

A Data Science & Analytics minor project that analyzes WhatsApp group chats to uncover communication patterns, user behavior, activity trends, and personality insights using Python.

---

## 📌 Project Overview

Group DNA is a WhatsApp Chat Analysis project developed using Python. The project reads exported WhatsApp chat data, cleans and processes it, and generates meaningful insights through data analysis and visualization.

The notebook performs multiple analytical features to understand how participants interact within a group.

---

## ✨ Features

### 🔹 Feature 1 – Chat Parser
- Reads WhatsApp exported chat file
- Parses date, time, sender, and message
- Detects:
  - System messages
  - Media messages
  - Deleted messages
- Creates a structured dataset

---

### 🔹 Feature 2 – Group Overview
Provides overall statistics including:
- Total messages
- Total participants
- Total words
- Media shared
- Deleted messages
- System messages

---

### 🔹 Feature 3 – Most Active Day & Hour
Analyzes:
- Most active day of the week
- Most active hour
- Daily message distribution
- Hourly activity trends

---

### 🔹 Feature 4 – Activity Heatmap
Visualizes group activity using:
- Day vs Hour Heatmap
- Peak chatting periods

---

### 🔹 Feature 5 – Top Words
Performs text analysis by:
- Removing stopwords
- Counting word frequencies
- Displaying the most commonly used words

---

### 🔹 Feature 6 – Response Speed & Silent Streaks
Measures:
- Average response time
- Longest inactive period
- Silent streak analysis

---

### 🔹 Feature 7 – Personality Archetype Detection
Identifies participant behavior based on:
- Message frequency
- Response activity
- Communication patterns

Example archetypes:
- Conversation Starter
- Active Member
- Silent Observer
- Fast Responder
